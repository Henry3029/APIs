A simple JavaScript project that allows users to search for GIFs using the Giphy API.

## Features
- Search for any GIF by keyword
- Fetches data from the Giphy API
- Displays a new GIF for each search
- Handles empty input and errors
- Beginner-friendly project

## Built With
- HTML
- CSS
- JavaScript (Vanilla JS)
- Giphy API

## How to Use
1. Type a word (e.g. `cats`, `football`, `anime`) into the input field  
2. Click the **Submit** button  
3. A related GIF will appear  

## Project Setup
Just open the `index.html` file in your browser.

No frameworks. No bundlers. Simple and clean.
